# Excellent Cutz - Professional Barbershop Website

Website profesional para la barbería **Excellent Cutz**, ubicada en Miami, FL.

## 📁 Estructura del Proyecto

```
peluqueria1/
├── index.html          # Página principal
├── styles.css          # Hoja de estilos
├── script.js           # JavaScript para interactividad
├── privacy.html        # Política de privacidad
├── cookies.html        # Política de cookies
├── favicon.svg         # Icono del sitio
├── images/             # Carpeta con imágenes del negocio
└── README.md           # Este archivo
```

## 🚀 Cómo Usar

### Opción 1: Abrir directamente
Simplemente haz doble clic en `index.html` para abrir la web en tu navegador.

### Opción 2: Usar un servidor local (recomendado)
Para una experiencia óptima, usa un servidor local:

**Con Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**Con Node.js (Live Server):**
```bash
npx live-server
```

**Con PHP:**
```bash
php -S localhost:8000
```

Luego abre `http://localhost:8000` en tu navegador.

## ✨ Características

### Diseño Responsivo
- Mobile first: optimizado para móviles, tablets y escritorio
- Se adapta a todos los tamaños de pantalla

### Bilingüe (Inglés / Español)
- Selector de idioma en el menú de navegación
- Cambio instantáneo sin recargar la página
- Preferencia guardada en localStorage

### Secciones
1. **Inicio (Home)**: Hero image, rating, introducción y botones de acción
2. **Servicios (Services)**: 6 servicios con iconos, descripciones y precios
3. **Galería (Gallery)**: Grid responsivo con imágenes del negocio y lightbox
4. **Contacto (Contact)**: Información, mapa, horario y formulario

### Elementos Adicionales
- 📱 Botón flotante de WhatsApp
- ⬆️ Botón "Subir arriba" (Back to top)
- 🖼️ Lightbox para galería de imágenes
- 📋 Formulario de contacto con validación
- 🔗 Navegación sticky con smooth scroll
- 📄 Páginas de políticas de privacidad y cookies

## 📱 Funcionalidades JavaScript

- **Selector de idioma**: Cambia entre inglés y español
- **Smooth scroll**: Navegación suave entre secciones
- **Navbar dinámico**: Cambia de color al hacer scroll
- **Lightbox**: Vista ampliada de imágenes de la galería
- **Formulario**: Validación y notificaciones
- **Back to top**: Botón para volver al inicio
- **Menú móvil**: Hamburguesa para dispositivos pequeños

## 🔧 Personalización

### Cambiar número de WhatsApp
Edita `index.html` y `script.js`, reemplaza `13055551234` por el número real.

### Actualizar información de contacto
Modifica los textos en `index.html` en la sección de contacto.

### Cambiar imágenes
Reemplaza las imágenes en la carpeta `images/` manteniendo los nombres de archivo.

### Modificar servicios
Edita las traducciones en `script.js` dentro del objeto `translations`.
